What does this mean?

The element box can be used to select a certain element, for example if you wanted to modify the Options Button, you would put in "#OptionsButton" as that's the ID for the element, here is a table of elements below:

Element ID
Home Button #BotButton
Config Button #ConfigurationButton
Market Button #MarketButton
Info Button #InformationButton
Debug Button #DEBUGButton
Options Button #OptionsButton
Every button .button

For the more techinal people here, this box uses a CSS selector which means you can select multiple things using classes, for example, all buttons that aren't at the side have a class called "button" so you can put in ".button" and select all of them.
